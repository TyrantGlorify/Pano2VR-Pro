# Installation Guide — Pano2VR Pro

## System Requirements
- Windows 10 or Windows 11 (64-bit)
- Administrator rights required for installation

## Steps
1. Download the installer package
2. Right-click the setup file and select **Run as administrator**
3. Accept the license agreement and choose the install directory
4. Follow the on-screen prompts to complete installation
5. Launch the application from the Start menu or desktop shortcut

## Notes
- An internet connection may be required on first launch
- Restart the system if prompted
- Antivirus software may flag the installer — add to exclusions if needed

## Uninstalling
Use **Control Panel → Programs → Uninstall a program** or the built-in uninstaller in the installation directory.
